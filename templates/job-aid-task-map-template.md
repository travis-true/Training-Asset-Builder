## Design Notes
* Use six rounded cards in a two-column grid.
* Use the primary brand color for the header.
* Use secondary brand color for section labels.
* Keep the core workflow in a horizontal banner.
* Use minimal icons with text labels.
* Do not include screenshots unless they are necessary for task completion.

## Accessibility Notes
* Do not use icons or colors without text labels.
* Keep all body text at least 10 pt.
* Use real text rather than converting the map to one flattened image.
* Add a logical reading order from top to bottom.
